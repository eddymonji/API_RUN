Description de l'api Run

L'api Casoft Run à pour objectif de nous familliariser avec les notions sur les serveurs web

Objectif de l’Api : Découvrir l'administration de serveurs sous Linux et le fonctionnement de la technologie Docker Objectifs spécifiques : Savoir utiliser un serveur sous Linux Savoir installer et configurer un serveur web, un serveur PHP, un serveur PostgreSQL Savoir utiliser la technologie de conteneurisation Docker Objectifs transversaux : Savoir lire la documentation technique Savoir gérer les erreurs système (compréhension, analyse, recherche)
